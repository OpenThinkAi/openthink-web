bring-your-own-view
CLI runtime.

Customizable browser views, on demand, for any CLI in any language. Mount a template once with ui-leaf mount; ship the data — get a rendered view in the browser. Node SDK + a language-neutral binary.

$npm install -g @openthink/ui-leaf
binary: ui-leaf · works from Bash, Python, Rust, Go — anywhere you can shell out
